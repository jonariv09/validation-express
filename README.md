Para ejecutar el proyecto se necesita un archivo *.env* con las siguientes variables 

- DB_PASS
- DB_USER
- DB_URI
- FACEBOOK_APP_ID
- FACEBOOK_APP_SECRET

**setting object for cookie**
{ path: '/', httpOnly: true, secure: false, maxAge: null }
